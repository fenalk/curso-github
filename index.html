os passos para criar um repositório no github

1. "git init" dentro da pasta do repositório
2. "git . add" adicionar um arquivo
3. "git commit -m 'first commit'" comitamos um arquivo
4. "git branch -M main" criar branch
5. "git remote add origin https:// ..." passa qual o repositorio que criamos no github
6. "git push -u origin main" para enviar o arquivo do nosso local para repositorio
7. "git pull" para adicionar arquivo do repositorio para o local
